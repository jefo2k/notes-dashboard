# Notes Board API

This is the back-end application that supports the Notes Board client.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Documentation

The project has a swagger end-point: [http://localhost:3000/api-docs/](http://localhost:3000/api-docs/).
